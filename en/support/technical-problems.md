  <Accordion title="What to do if Tooli doesn't load?">
    Try:

    1. Refreshing the page
    2. Logging out and back in

    Still stuck? Contact our support team via [support@meetwonka.com](mailto:support@meetwonka.com)
  </Accordion>

  <Accordion title="What file types does Tooli support?">
    Tooli accepts the following file types:

    * PDF
    * Microsoft Word (DOC, DOCX)
    * Microsoft Excel (XLS, XLSX)
    * Microsoft PowerPoint (PPT, PPTX)
    * Plain text (TXT)
    * Images (PNG, JPG, JPEG)
  </Accordion>

  <Accordion title="Are there any upload limits?">
    Maximum file size: 10 MB per file

    Maximum files per message: 10
  </Accordion>

  <Accordion title="Why can't I upload my file?">
    Common reasons include:

    - The file is larger than 10 MB -- try to compress or split large files
    - The file is password-protected or encrypted -- try to remove restrictions
    - The file type is not supported -- convert unsupported files
  </Accordion>

  <Accordion title="My handwritten text is scanned and Tooli doesn't read it well">
    The AI that Tooli uses can read scanned documents quite well, but quality matters.

    For better accuracy:

    - Use a clear, well-lit scan
    - Make sure the text is readable and not distorted -- if the document is hard to read for a human, Tooli may struggle too.
  </Accordion>

  <Accordion title="Can I upload photos from my phone?">
    Definitely! For the best response, make sure:

    - The photo is stable and well lit
    - Any text is not too small

    You can also add some written/typed information with your pictures, this can boost the response often significantly.
  </Accordion>
</AccordionGroup>
